Mini-Projet

🍄 Bienvenue sur le mini-projet. 🍄

À propos :

Ce mini-projet a pour but de tester des frameworks comme Bulma, d’écrire des scripts Bash, et de renforcer notre compréhension du web scraping en ligne de commande. Ces exercices pratiques nous permettent de nous préparer pour le projet collaboratif intitulé "La vie des mots sur le web" qui vise à étudier l’usage d’un mot et ses "variations" en plusieurs langues sur le web. Nous explorerons comment les mots polysémiques changent de sens selon leur contexte d’utilisation et entre différentes langues.

Le tableau en question :

Ci-dessous, vous pouvez consulter un exemple de tableau que j’ai généré en tant que test à partir de mon script Bash. Ce tableau illustre comment les URLs devraient fonctionner dans le cadre de mon projet. Le script automatise la tâche de création de tableaux organisés pour notre travail de web scraping.
